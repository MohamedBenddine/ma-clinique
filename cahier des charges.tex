\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

\section*{English}
This project introduces the development of a web application for medical appointment booking called "Ma Clinique". The main objective is to reduce long queues in hospitals and improve healthcare access for people living in rural areas. The application allows patients to view doctors' availability and book convenient time slots. It also provides doctors with an efficient appointment management system.

The system uses a modern web architecture based on PHP, MySQL, and Bootstrap, with multilingual support (English/Arabic) to ensure better accessibility. Main features include online appointment booking, appointment status verification, medical schedule management, and a comprehensive dashboard for healthcare professionals.

This solution contributes to the digitalization of the healthcare sector and improves organizational efficiency of medical facilities while reducing administrative burden.

\chapter{Introduction}

\section{Project Context}

In an increasingly digitalized world, healthcare services must adapt to new technologies to improve the efficiency and accessibility of care. Traditional medical appointment management, based on phone calls and in-person visits, presents numerous challenges, particularly in rural areas where access to healthcare is limited.

The "Ma Clinique" project is part of this healthcare modernization effort by proposing an innovative web solution for medical appointment booking. This application aims to simplify and optimize the reservation process for both patients and healthcare professionals.

\section{Problem Statement}

The current healthcare system faces several major challenges:
\begin{itemize}
    \item Long waiting queues in healthcare facilities
    \item Difficulties accessing care for rural populations
    \item Inefficient management of medical schedules
    \item High rate of missed appointments
    \item Administrative overload of medical staff
    \item Limited communication between patients and doctors
\end{itemize}

\section{Proposed Solution}

The "Ma Clinique" web application offers a comprehensive medical appointment management platform that enables:
\begin{itemize}
    \item Online appointment booking 24/7
    \item Real-time availability visualization
    \item Automated schedule management
    \item Appointment tracking for patients
    \item Complete dashboard for doctors
    \item Multilingual support (English/Arabic)
\end{itemize}

\section{Scope}

\subsection{Project Scope}
This project covers the following aspects:
\begin{itemize}
    \item Development of a web interface for appointment booking
    \item Creation of a management system for doctors
    \item Implementation of a database to store information
    \item Development of a multilingual system (English/Arabic)
    \item Creation of responsive interfaces for different devices
    \item Implementation of basic security measures
    \item User and technical documentation
\end{itemize}

\subsection{Out of Scope}
The following elements are not included in this version:
\begin{itemize}
    \item Online payment system
    \item Automatic SMS or email notifications
    \item Native mobile application
    \item Integrated telemedicine
    \item Integration with existing hospital systems
    \item Complete electronic medical records
    \item Automatic reminder system
\end{itemize}

\chapter{System Analysis}

\section{Problem Description}

The healthcare system in several regions, particularly rural areas, suffers from poor medical appointment management. Patients often have to travel to make an appointment, which can be constraining and inefficient. This generates endless queues, delays, or even lost appointments. Moreover, medical resources are not used optimally, which affects the quality of care.

Given this situation, it becomes essential to implement a solution that facilitates remote appointment booking and improves coordination between patients and healthcare professionals.

\section{Project Objectives}

\subsection{Primary Objectives}
\begin{itemize}
    \item Reduce waiting queues in health centers
    \item Facilitate appointment booking for patients, especially those living in rural areas
    \item Provide doctors with a simple and efficient appointment management tool
    \item Improve overall organization of medical consultations
    \item Reduce missed appointments (no-shows) through automatic reminders
\end{itemize}

\subsection{Secondary Objectives}
\begin{itemize}
    \item Digitize the appointment booking process
    \item Improve patient satisfaction
    \item Optimize the use of medical resources
    \item Provide statistics to improve management
    \item Ensure a multilingual interface for better accessibility
\end{itemize}
